| ------------------ | ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| `system_configs`   | Menyimpan konfigurasi default global sistem (hanya satu record) | `selected_asr`<br>`selected_tts`<br>`selected_llm`<br>`selected_vlllm`<br>`prompt`<br>`quick_reply_words`<br>`delete_audio`<br>`use_private_config` | Modul yang digunakan secara default (ASR, TTS, LLM, VLLLM)<br>Prompt default<br>Kata balasan cepat (JSON)<br>Apakah menghapus audio<br>Apakah menggunakan konfigurasi pribadi | Digunakan untuk pengaturan default global    |
